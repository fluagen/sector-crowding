# 📊 板块拥挤度 (Sector Crowding)

**A 股板块资金情绪评估工具** —— 实时分析任意板块的资金拥挤度、主力资金流向与量价关系，识别阶段性顶部风险。

板块拥挤度衡量的是某一个板块/题材吸走整个市场流动性的程度。当热钱过度集中于单一板块，往往意味着阶段性顶部正在形成。这个 Claude Code 自定义 Skill 将这套分析框架自动化，几秒钟内输出结构化风险评估报告。

---

## 核心能力

基于 **三支柱分析框架**，综合三个维度交叉验证：

| 支柱 | 核心问题 | 关键指标 |
|------|---------|---------|
| **拥挤度比率** | 这个板块吸走了市场多少流动性？ | 板块成交额 / 全市场成交额 × 100% |
| **资金净流入** | 大资金在进场还是在撤退？ | 主力资金净流入/流出（亿元） |
| **量价背离** | 成交量和价格走势是否健康？ | 量增价涨 / 量增价跌 / 量缩价涨 |

> ⚠️ **关键原则：比率和净流入必须一起看。** 同样的 16% 拥挤度，搭配 212 亿净流入 vs 59 亿净流入，结论天差地别。

## 它能做什么

- ✅ **实时计算拥挤度**：自动搜索东方财富、同花顺等数据源，计算任意板块的实时拥挤度比率
- ✅ **追踪主力资金**：判断机构资金是净流入还是净流出，识别资金偏好的持续性变化
- ✅ **识别量价背离**：检测放量冲高回落、量增价平等顶部信号
- ✅ **风险等级评估**：基于决策矩阵输出 🟢🟡🟠🔴 四级风险评级
- ✅ **历史对比**：与历史极端节点（如半导体 2025.5.20 的 16.1%）对比当前水平
- ✅ **可操作建议**：分别对持仓者和空仓者给出具体建议和关键观察点
- ✅ **置信度标注**：数据完整时标注高置信度，数据缺失时诚实降级，绝不捏造数据

## 分析框架详解

### 拥挤度比率阈值

| 比率区间 | 含义 | 历史参考 |
|---------|------|---------|
| < 10% | 正常 | 板块催化期: 8.5% |
| 10-14% | 偏高 | 2026年1月行情: 11.3% |
| 14-16% | 高位 | 接近极端区域 |
| > 16% | 极端 | 半导体 2025.5.20: 16.1% / 2025.6.25: 16.5% |

> 阈值是相对的——小市值概念题材可能在 10-12% 就已极端。

### 量价背离信号

| 信号 | 判断 |
|------|------|
| 量增价涨 | 量价配合，趋势健康 |
| 量增价平 | 初步背离，警惕 |
| 量增价跌（放量冲高回落） | **强烈见顶信号** |
| 量缩价涨 | 动能衰减 |

### 数据采集策略

分阶段并行搜索：东方财富 → 新浪财经 → 同花顺 → 雪球，数据缺失时自动降级并标注置信度。

## 历史案例：半导体板块

| 日期 | 拥挤度 | 净流入 | 结论 |
|------|--------|--------|------|
| 2025.5.20 | 16.1% | -- | 比率创新高 → 警示。次日高开 1% 后收跌 6% |
| 2025.6.24 | 16.4% | +212 亿 | 动能强劲，可享受鱼尾但保持警惕 |
| 2025.6.25 | 16.5% | +59 亿 | 流入骤降，警报拉响！机构开始撤退 |

**教训：** 相同的高比率，不同的净流入趋势，结论完全不同。不能只看一个指标。

## 原理与局限性

### 这不是择时系统，这是预警系统

- 拥挤度到达极端 ≠ 明天一定跌。它告诉你的是：**收益风险比已经恶化**，而非"立即卖出"
- 牛市容许更高的拥挤度，熊市中危险阈值更低
- 必须结合基本面、政策面、市场情绪等多维度综合判断

### 方法论哲学

> 不预测方向，只做应对方案。如果继续涨 → 持有；如果冲高回落 → 兑现。
> 当所有散户都相信某板块就是当下的真神，就到了机构派发筹码的时刻。
