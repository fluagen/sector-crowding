- 全市场 = 沪市主板 + 深市主板 + 创业板 + 科创板

# 板块拥挤度 (Sector Crowding)

## Overview

板块拥挤度衡量的是某一个板块/题材吸走整个市场流动性的程度。当热钱过度集中于单一板块，往往意味着阶段性顶部正在形成。

## Three-Pillar Analysis Framework

### 1. 拥挤度比率 (Crowding Ratio)

```
拥挤度比率 = 板块成交额 / 全市场成交额 × 100%
```

**参考阈值（来自历史数据）：**

| 比率区间 | 含义 | 历史参考 |
|---------|------|---------|
| < 10% | 正常 | xxx板块/题材 催化期 (2025.2): 8.5% |
| 10-14% | 偏高 | xxx板块/题材 2026年1月行情: 11.3% |
| 14-16% | 高位 | 接近极端区域 |
| > 16% | 极端 | xxx板块/题材 2025.5.20: 16.1%; 2025.6.25: 16.5% |

注意：阈值是相对的。不同板块的市场容量不同，16%对于任何一个单一板块都是极端信号。小板块/题材的阈值需要酌情降低。

### 2. 资金净流入 (Net Capital Inflow)

追踪当日板块主力资金是净流入还是净流出（单位：亿元）。

- **强劲流入**（如 > 150亿）：上涨动能充足，即使比率高也可能维持
- **流入放缓**：高比率下流入骤降是危险信号
- **净流出**：机构派发筹码的经典迹象
- **资金偏好**：连续3天资金持续流入该板块，资金对该板块偏好增加，需重点关注该板块机会

**关键组合判断：**
- 比率 16.4% + 净流入 212亿 → 动能强劲，继续持有 (2025.6.24 案例)
- 比率 16.5% + 净流入仅 59亿 → 风险急剧上升，收益风险比恶化 (2025.6.25 案例)
- 比率 5% + 资金偏好 → 资金偏好增加，重点关注该板块的机会

### 3. 量价背离 (Volume-Price Divergence)

| 信号 | 判断 |
|------|------|
| 量增价涨 | 量价配合，趋势健康 |
| 量增价平 | 初步背离，警惕 |
| 量增价跌（放量冲高回落） | 强烈见顶信号 |
| 量缩价涨 | 动能衰减 |

**关键观察点：** 开盘高开后放量回落、大单砸出但无大单承接 = 机构在散户接盘时出货。

## Data Collection Protocol

### 分阶段搜索策略

**Phase 1 — 并行搜索（同时发起 4 个方向）：**

```
搜索1: "[sector_name] 板块 今日成交额 资金流向"
搜索2: "A股 两市成交额 今日"
搜索3: "[sector_name] 板块 资金净流入 主力资金"
搜索4: "[sector_name] 板块 成交额占比 历史最高 拥挤度"
```

**Phase 2 — 深度抓取（选取 2-3 个最有价值的结果）：**

```
WebFetch(prompt: "提取：板块今日成交额、涨跌幅、资金净流入金额、主力净流入")
```

**优先数据源：** eastmoney.com > finance.sina.com.cn > 10jqka.com.cn > xueqiu.com

**Phase 3 — 缺口补全（条件触发）：**

如果 Phase 1+2 后仍有必要数据缺失，进行 1-2 次定向补充搜索。无法获取的数据在报告中明确标注，并下调置信度。

### 板块分类

#### 申万行业分类

适用于行业板块、指数板块：

- 周期：煤炭、钢铁、有色、石油石化、基础化工、公用事业
- 科技（TMT）：电子、计算机、通信、传媒、电力设备、军工
- 消费：食品饮料、医药生物、家电、美容护理、商贸零售、农林牧渔
- 金融金融：银行、非银金融、房地产

#### 题材 / 概念板块

同花顺、东方财富、通达信自研，跨传统行业：

- 产业赛道概念：AI 算力、半导体、光伏储能、新能源车、创新药、信创、光模块、六氟化钨、磷化铟等
- 政策题材：国企改革、一带一路、数字经济、军工、乡村振兴
- 资金属性：北向重仓、两融标的、高股息、ST、次新股
- 事件题材：重组、并购、算力租赁、MR、人形机器人

### 数据降级策略

- **实时数据完整** → 完全分析，置信度高
- **延迟 1-2 天** → 标注日期，置信度中
- **部分数据可用** → 用可用指标分析，标注缺失项
- **无数据** → 报告"数据不足"，建议用户手动查看来源

## Decision Matrix

根据三个指标的综合判断：

| 拥挤度比率 | 净流入趋势 | 量价关系 | 风险等级 | 建议 |
|-----------|-----------|---------|---------|------|
| 正常/偏低 | 强劲流入 | 量价配合 | 🟢 低风险 | 可考虑参与，板块未拥挤 |
| 正常/偏低 | 流入放缓 | 背离初现 | 🟡 中低风险 | 持有观察，关注恶化信号 |
| 接近历史高位 | 强劲流入 | 量价配合 | 🟡 中风险 | 持有，动量充足但需警惕 |
| 接近历史高位 | 流入骤降 | 中性 | 🟠 高风险 | 风险大于收益，准备兑现 |
| 创新高 | 流入骤降 | 量价背离 | 🔴 极端风险 | 强烈卖出信号，落袋为安 |
| 创新高 | 强劲流入 | 量价配合 | 🟠 中高风险 | 可享受鱼尾，但加100个心眼 |

## Example Walkthrough: 半导体板块

**2025年5月20日 — 极端拥挤，警示发出**

| 指标 | 数值 |
|------|------|
| 半导体成交额 | 4751亿 |
| 全市场成交额 | 2.95万亿 |
| 拥挤度比率 | 16.1%（当时历史新高）|

结论：比率创历史新高 → 警告信号。次日（5月21日）半导体高开1%后收跌6%。

**2025年6月24日 — 极端拥挤但动能充足**

- 比率 16.4%（又创新高），净流入 212亿（也创新高）
- → 动能维持，享受鱼尾但保持警惕

**2025年6月25日 — 极端拥挤且流入崩溃**

- 比率 16.5%（再创新高），成交额 5930亿，净流入仅 59亿
- → 警报拉响！机构开始撤退，风险 >> 收益
- → 建议：加100个心眼，警惕放量冲高回落

**教训：** 相同的高比率，不同的净流入趋势，结论完全不同。不能只看一个指标。

## Common Mistakes

| 错误 | 纠正 |
|------|------|
| 只看比率不看净流入 | 6月24日（16.4%+212亿）vs 6月25日（16.5%+59亿）结论天差地别 |
| 把 16% 当成放之四海皆准的绝对阈值 | 小板块（如某个概念题材）可能在 10-12% 就已极端 |
| 预测"明天必跌" | 方法论只描述风险收益的非对称性，不提供择时信号 |
| 忽视市场整体环境 | 牛市和熊市的拥挤度含义不同，牛市容许更高拥挤度 |
| 只看一天数据做判断 | 关注趋势变化（连续数日的比率和流入方向）比单日数值更重要 |

## Guiding Principles

1. **不预测方向，只做应对方案。** 如果继续涨→持有；如果冲高回落→兑现。
2. **拥挤度是预警系统，不是择时系统。** 比率到极端≠明天跌，但意味着收益风险比已经恶化。
3. **当所有散户都相信某板块就是当下的真神，就到了机构派发筹码的时刻。**
4. **永远输出免责声明。** 这是分析工具，不是投资顾问。
5. **数据缺失时诚实标注，绝不捏造数据。**
